1. PUT 用于创建或完全替换文档

PUT 方法要求必须提供文档 ID，用于创建或完全替换已有文档。如果指定的文档 ID 不存在，则 PUT 会创建一个新文档；如果该 ID 已存在，则会完全覆盖原有数据，不会保留任何旧字段。
