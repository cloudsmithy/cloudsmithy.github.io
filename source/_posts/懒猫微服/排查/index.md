踩坑记录与问题排查，希望我踩过的坑你不用再踩。

排查篇收录的是我在使用懒猫微服过程中遇到的各种问题和解决方案。这些问题有的是懒猫微服特有的，有的是 Docker 或 Linux 通用的，但都是在懒猫微服这个环境下遇到并解决的。

第一篇记录的是应用商店 pip 使用清华源报错 403 Forbidden 的问题。这是一个很诡异的问题，同样的清华源地址，在本地能用，在懒猫微服的应用容器里就报 403。经过一番排查，最终找到了原因和解决方案。这篇文章详细记录了整个排查过程，包括如何查看容器日志、如何定位问题、以及最终的解决方法。

第二篇是关于上架应用后 pip 安装报错 HASH 不一致的问题。这个问题发生在应用打包上架之后，用户安装时 pip 报错说包的 HASH 值和预期不符。问题的根源和解决方案都在文章里，如果你也在开发懒猫应用，这篇文章值得一看。

第三篇记录的是一个网络相关的问题：Mac 上 Orbstack 的 Docker 容器访问懒猫微服上的 WordPress 时，出现 IPv6 解析问题。这个问题比较小众，但如果你正好遇到了类似的情况，这篇文章可能会帮到你。

排查篇的文章数量不多，因为不是所有问题都值得写成文章。我收录的都是那些排查过程比较曲折、解决方案不太容易找到的问题。每一篇都包含了完整的问题描述、排查思路和最终方案，希望能帮你节省排查时间。

如果你在使用懒猫微服的过程中遇到了什么奇怪的问题，也欢迎和我交流。说不定你的问题会成为排查篇的下一篇文章。
