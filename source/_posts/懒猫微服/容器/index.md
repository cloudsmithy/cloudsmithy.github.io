title: 📦 容器篇
date: 1970-01-01 08:00:00
---

写给懒猫微服玩家的 Docker 小书，用故事的方式讲解容器基础。

容器篇是一个特别的系列。它不是传统的技术教程，而是用故事的形式来讲解 Docker 的核心概念。如果你之前对容器一无所知，或者看过一些教程但还是似懂非懂，这个系列会给你全新的理解方式。

第一篇《无法部署的诅咒》从一个常见的问题开始：为什么同样的代码，在我电脑上能跑，到服务器上就不行了？这个问题引出了容器技术诞生的背景和它要解决的核心问题。读完这篇，你会理解为什么我们需要 Docker。

第二篇《镜像旅馆的秘密》讲的是 Docker 镜像。镜像是容器的基础，但它到底是什么？和虚拟机镜像有什么区别？为什么镜像可以分层？这篇文章用"旅馆"的比喻，把这些概念讲得通俗易懂。

第三篇《容器部落生活》深入容器的运行时。容器启动后发生了什么？容器之间如何隔离？容器和宿主机是什么关系？这篇文章会带你走进容器的内部世界。

第四篇《数据之岛与持久化卷》解决一个关键问题：容器是临时的，但数据需要持久保存，怎么办？这篇文章详细介绍了 Docker 的数据卷机制，以及各种数据持久化的最佳实践。

第五篇《多容器交响曲：Docker Compose 上场》是系列的收官之作。现实中的应用往往需要多个容器协作，比如 Web 服务器 + 数据库 + 缓存。Docker Compose 就是用来编排多容器应用的工具，这篇文章会教你如何使用它。

容器篇的五篇文章形成了一个完整的学习路径，从"为什么"到"是什么"再到"怎么用"，循序渐进。读完这个系列，你就具备了在懒猫微服上玩转 Docker 的基础知识。
