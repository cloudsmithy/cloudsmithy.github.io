如果你刚拿到懒猫微服，或者正在考虑入手，这个系列就是为你准备的。

入门篇是整个专栏的基石，目前已经更新了 30+ 篇实战教程，覆盖了从开箱到日常使用的方方面面。每一篇都是我亲自踩坑、反复验证后写下的保姆级教程，力求让零基础的用户也能跟着操作。

这个系列从最基础的开箱开始，带你认识懒猫微服的硬件配置和系统架构。然后是 SSH 远程登录的配置，这是玩转懒猫微服的第一步，也是后续所有进阶操作的基础。我会手把手教你如何开启 SSH、配置免密登录，让你随时随地都能连上自己的设备。

接下来是网盘和文件共享相关的内容。懒猫网盘是懒猫微服的核心功能之一，我会教你如何通过 SMB 协议挂载网盘、如何在电视盒子上播放视频、如何用 Time Machine 把 Mac 备份到懒猫微服。这些都是日常使用中最高频的场景。

网络配置也是入门必修课。如何设置静态 IP、如何接入旁路由实现科学上网、如何解决 DNS 和本地代理的冲突，这些问题我都遇到过，也都找到了解决方案。

除此之外，入门篇还涵盖了大量实用应用的部署教程：用 2fauth 共享 MFA 验证码、用 FreshRSS 订阅 RSS、用 Memos 替代 Flomo 做笔记、用 Vaultwarden 替代 1Password 管理密码、用 planka 做项目管理...每一个应用都是我精挑细选、亲测好用的。

如果你是开发者，入门篇也有专门为你准备的内容：如何把懒猫微服当作 24×7 在线开发机、如何用 SSH Remote 打造云端开发环境、如何给懒猫安装图形界面实现远程桌面。

总之，入门篇的目标是让你快速上手懒猫微服，把它变成你生活和工作中不可或缺的数字基础设施。每一篇文章都能省你一天的折腾时间，建议按顺序阅读，也可以根据自己的需求跳着看。
